Gestion des règle de type MIME

Cette fonctionnalité vous permet de paramétrer les filtres par type MIME. Ainsi vous pouvez autoriser ou interdire le dépôt de fichiers d’un type donné, e.g. interdire le dépôt de fichiers vidéos (MPEG, etc.).

Une fois activée :

  • Depuis le menu de navigation, allez à la rubrique « Paramètres > Règles de type MIME » ;
  • Sélectionnez un domaine depuis l’« Arbre des domaines », cliquer sur le bouton « Ajouter » ;
  • Cliquer sur le filtre « Mon filtre » ;
  • Pour chaque type MIME, activez-le ou désactivez-le, puis enregistrez en cliquant sur le bouton « Soumettre ».
