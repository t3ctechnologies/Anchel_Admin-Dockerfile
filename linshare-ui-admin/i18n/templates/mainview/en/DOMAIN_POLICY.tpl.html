Cross-domain communication rules

Inter-domain communication rules make it possible to manage communication (list / share) between users in different domains or within the same domain.
When you create a domain through the view 'Domain configuration' and you choose the option 'Automatic creation' for 'Interdomain communication rules', a rule will be created automatically by authorizing the domain         newly created to communicate with him only. All other communications are prohibited by default.
For a rule to be taken into account, it must be assigned to a domain.
