Pied de courriel

Cette vue vous permet de créer des templates pour les pieds de courriel. Il vous faudra créer un modèle HTML & Text. Dans la vue 'Configuration de courriel' vous pouvez crées des modèles de structure de courriel et associer une structure de courriel avec un pied de courriel (Attention: La structure de courriel doit posséder la variable ${footer} afin de pouvoir y insérer le pied de courriel).
