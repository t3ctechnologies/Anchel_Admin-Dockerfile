Management of MIME type rules

This view allows you to set the filters by MIME type. So you can allow or prohibit the filing of files of a given type, e.g. prohibit the filing of video files (MPEG, etc.) Operation :

  • Select a domain from the "Domain Tree", click the "Add" button and create your filter;
  • Then click on the filter of your choice;
  • For each MIME type, enable or disable it, and save by clicking the "Save" button.
