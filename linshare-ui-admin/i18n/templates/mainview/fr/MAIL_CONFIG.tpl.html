Configuration de courriel

Cette vue vous permet de créer des structures de courriel pour vos domaines. À partir des informations saisies dans les vues :

  • 'Structure de courriel' permet de créer des modèles de structure de courriel, des mises en pages différentes
  • 'Pied de courriel' pour la mise en page du pied de courriel
  • 'Contenu de courriel' pour gérer le contenu des notifications envoyées automatiquement aux utilisateurs de l'application

Après avoir paramétré vos différents modèles de structure de courriel vous pourrez, dans la vue 'Configuration des domaines' (lors de la création / édition d'un de vos domaines), choisir une configuration de courriel.
