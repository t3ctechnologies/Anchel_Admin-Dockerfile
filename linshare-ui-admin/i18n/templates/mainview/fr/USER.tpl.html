Liste des utilisateurs

Cette vue vous permet d'effectuer une recherche parmi les utilisateurs des différents domaines. Vous pourrez ensuite éditer les rôle, prénom, nom et langues par défaut, ainsi que les droits de dépôt de l'utilisateur sélectionné.Fonctionnement :

  • Vous devez saisir dans un des champs de filtre les données de l'utilisateur souhaité.
  • Cliquez sur l'utilisateur pour accéder à sa fiche personnelle.
