Rapport d'exécution d'une tache de migration.

Le rapport d'exécution d'une tache de migration est consignée ci-dessous. La console vous permet de consulter les logs d'activité de celle-ci.

Attention: Il s'agit de logs simplifiés. Pour avoir le détail avancé en cas d'erreur, il faudra vous référer aux logs du serveur Tomcat. Une bonne pratique est de mettre l'application Anchel en mode debug avant la migration.
