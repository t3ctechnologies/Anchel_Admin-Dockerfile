Personnalisez un message d'accueil pour les utilisateurs connectés à l'application 'Anchel'.Vous pouvez créer plusieurs messages d'accueil par domaines et les associer aux différents sous-domaines.Pour créer un message d'accueil il vous suffit de dupliquer un message d'accueil existant, pour ce faire :

  • Dans l'arbre des domaines de gauche vous trouverez la liste des domaines : parents (en lecture seul) et la liste de vos domaines et sous-domaines (en écriture).
    Vous pouvez dupliquer le message d'accueil grâce au bouton 'dupliquer' dans la liste des messages d'accueil ou en pré-visualisant le message d'accueil. Une fenêtre s'ouvrira suite à la demande de duplicât, vous devrez y saisir un nom et le domaine ou vous souhaitez copier le message d'accueil.
  • Grâce au bouton 'ajouter', une fenêtre s'ouvrira pour vous proposer de choisir un domaine afin d'y sélectionner un modèle de message d'accueil, vous devrez choisir un nouveau nom et une copie s'effectuera dans le domaine courant.

Suite à cette création vous pourrez modifier le message d'accueil et affecter celui aux domaines de votre choix qui pourront en bénéficier, c'est-à-dire, lors de la configuration des domaines vous pourrez y choisir un message d'accueil, la liste proposée seront les messages d'accueil affectés pour le domaine. Par défaut si aucun message d'accueil ne lui est afféctée, la liste ne proposera que le message d'accueil par défaut.
